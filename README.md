# app-pwa-tetris

HTML/JavaScript で作られたテトリスゲームの PWA アプリです。オフラインでもプレイ可能で、スマートフォンへのインストールにも対応しています。

## 機能

- テトリスの全7種テトリミノ対応
- スコア・レベル・ライン数の表示
- レベルアップによる落下速度の増加
- 次のピースのプレビュー
- 一時停止機能
- PWA 対応（オフラインプレイ・ホーム画面へのインストール）
- モバイル対応（タッチ操作ボタン付き）

## 操作方法

### キーボード（PC）

| キー | 操作 |
|------|------|
| `←` `→` | 左右移動 |
| `↑` | 回転 |
| `↓` | 落下速度アップ |
| `Space` | 即時落下 |
| `P` | 一時停止 |

### タッチ（スマートフォン）

画面下部のボタンで操作します。

## ファイル構成

```
tetris/
├── index.html          # メインゲームファイル
├── manifest.json       # PWA マニフェスト
├── sw.js               # Service Worker
├── icon-192.png        # アイコン (192x192)
├── icon-512.png        # アイコン (512x512)
├── icon-maskable-192.png  # マスカブルアイコン (192x192)
└── icon-maskable-512.png  # マスカブルアイコン (512x512)
```
